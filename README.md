# 1. 项目介绍
实现语音转SRT格式字幕，提供HTTP接口

# 2. 输出SRT格式字幕
示例：
```
1
00:00:02,810 --> 00:00:03,330
多谢

2
00:00:04,889 --> 00:00:06,009
请入席吧

3
00:00:24,530 --> 00:00:25,730
曹操谢座

4
00:00:26,969 --> 00:00:27,510
列位

5
00:00:28,989 --> 00:00:33,465
请
```

# 3. 运行方法
1. 安装依赖
```
uv sync
```
2. 运行
```
uv run main.py
```
